◆ parseFile()

bool liquid::Settings::parseFile ( std::string  file)
overridevirtual

Parses a file and assign Settings to values.

Parameters
fileFile to be parsed
Returns
True if parsed correctly, False otherwise

Loads appropriate data using inherited ConfigurationParser into the mData variable, then uses those loaded values to assign the hard values to each stored variable in this class. The class may be extended to hardcode more settings, otherwise you can retrieve them directly by using the functions provided by ConfigurationParser to access them.

Reimplemented from liquid::ConfigurationParser.
