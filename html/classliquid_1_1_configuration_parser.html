Compiles the given collection of lines into a std::map.

Parameters
linesCollection of lines you want to compile

Takes the given collection of lines and methodically parses through them creating tokens where the first token is the name of the key and the second value after whitespace is the value of the key.

◆ parseFile()

bool liquid::ConfigurationParser::parseFile ( std::string  file)
virtual

Parses the given file into paired <string, string> data.

Parameters
fileName of the file appended to the path of the file, relative to game executable
Returns
True if the file was parsed correctly, False otherwise

Takes the file at the given location and parses each line into a std::vector, that vector is then passed to ConfigurationParser::compileData(lines) which puts the data into mData as <key, value> pairs.

Reimplemented in liquid::Bindings, and liquid::Settings.
